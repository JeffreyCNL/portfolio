Project Distributed System

UW CSE452 course Project. The GitLab repo is private.

  • Implemented remote procedure call (RPC) interface supporting key-value store for many clients to a server.
  • Designed a key-value primary-backup replication system for continuation of operation through server failures.
  • Developed Paxos management of the key-value store, providing a majority of servers remain functional.
  • Integrated sharding of the key-value store across server groups with dynamic load balancing .
