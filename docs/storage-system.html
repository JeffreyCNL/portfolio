GARMIN Storage System

GARMIN Taiwan was trying to store their materials effectively. Our team built a production line consisted of 4 machines for them. This project end up earning over $300,000 for our company, which is the most profitable case in 2018 fiscal year.

  • Designed the first stop of the production line which was in charge of scanning barcode to determine which storage tower to store the materials.
  • Built primary layout to discuss with potential outlook and basic functionality with client.
  • Negotiated with other teams, e.g software / firmware team, electrical team with prospective design approach.
  • Troubleshoot with ill-fitted component and modify the design.
  • Skill related: Machine Design / Interpersonal skill / Multidisciplinary teams negotiation
